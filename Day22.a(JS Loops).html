Loops :-  
        To Perform Same Tasks Repeatedly


        3 Types:-
         While Loop:- Check Statements And Then Executes Untill False. Not Executed Untill Statement Is True.

            While Syntax:-
               while(Condition){
                  Statements
                  Increase/Decrement
               }





         Do While Loop:- Executes Statement First And Then CHecks The Statements.

             While Syntax:-
             Initialization
              do{
                  Statements
                 Increase/Decrement
              }
              while(Condition)






         For Loop:- To Repeat Task Untill Certain Desired Loop Number Is Reached.
              The Program Is Only Executed Untill The Statement Is True

              For Syntax:-
              for(Initialization; Condition; Increment/Decrement) 
              {
              STATEMENT
              }


           NOTE:-   ONLY THEAE AND NOT MORE THAN THESE I.E. ONLY FOR ONE VARIABLE
                     NO PARENTHESIS




           ALTERNATIVELY:-

           INITIALIZATION 
           FOR(CONDITION){
              STATEMENT()
              INCREMENT/DECREMENT
           }
